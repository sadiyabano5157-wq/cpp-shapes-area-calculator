# cpp-shapes-area-calculator
A simple C++ OOP project that demonstrates creating Rectangle and Circle classes, creating objects, storing dimensions, and calculating the area of different shapes.


# Shapes Area Calculator in C++

## Description
This project demonstrates basic Object-Oriented Programming concepts in C++.

The program:
- Creates Rectangle and Circle classes
- Creates objects of both classes
- Stores dimensions in objects
- Calculates the area of a rectangle and a circle
- Displays the results using cout

## Classes Used

### Rectangle
- length
- width

### Circle
- radius

## Formula Used
Rectangle Area = length × width

Circle Area = 3.14 × radius × radius

## Output
Length of Rectangle : 10
Width of Rectangle : 5
Area of Rectangle : 50

Radius of Circle : 7
Area of Circle : 153.86
